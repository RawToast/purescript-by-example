# PureScript exercises

Exercises from the book [PureScript by example](https://leanpub.com/purescript/read)

## Installing PureScript

npm install -g purescript pulp bower

* Npm dependencies:
  * `yarn`
* Fetching purescript dependencies:
  * `bower install`
* Build / Test
  * `pulp --watch build`
  * `pulp test`
* Repl
  * `pulp repl`
  * `./repl.sh`
